Welcome, interviewee person!

This is the Minted coding interview playground.

Shake off the nerves!— nothing crazy in here.

Our hope is that this project provides enough semi-realistic coding work to be able to explore your way of thinking about software and to work through some of it together.

To get started, click the "preview" URL above the terminal below, which will open the running application in your browser. The page will automatically reload on changes.

Coding exercises live in the interview-playground/src/playgrounds directory. Browse using the Projects Explorer to the left.

If the development server and/or webpack watch process die for whatever reason, they can be re-started with the play button up top.

Have fun!
