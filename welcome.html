Welcome, interviewee person!

This is the Minted coding interview playground.

Shake off the nerves!— nothing crazy in here.

To get started, click the "preview" URL in the "app" tab below, which will open the running application in your browser. The page will automatically reload on changes. To view the test runner (useful for some exercises), open the "tests" tab.

Coding exercises live in the interview-playground/src/playgrounds directory. Browse using the Projects Explorer to the left.

If the development server dies for whatever reason, it can be re-started with the play button up top. Both the dev server and the tests can be re-started from the "Commands" explorer to the left.

Have fun!
